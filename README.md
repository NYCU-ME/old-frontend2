# NYCU-ME-frontend

## 架構

`src/App.vue` 是主要應用程式，他會依據 URL 的不同，從 `src/router/index.js` 中選擇不同的 View 給 Render 出來。
所有的設定都在 `src/config.json` 當中。

## Project setup

```
npm install
```
